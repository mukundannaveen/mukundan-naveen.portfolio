My Skills

HTML5

HTML is the backbone of web pages. It structures the content using elements like headings, paragraphs, images, and links. Every website starts with HTML.

CSS

CSS3 (Cascading Style Sheets Level 3) is the latest standard of CSS used to style and layout web pages. It introduced powerful features that enhanced the look and feel of websites without relying heavily on JavaScript or images.

JavaScript

JavaScript is a scripting language used to create interactive features on websites—like sliders, form validations, dynamic content updates, and more. It runs in the browser.

React

React is a JavaScript library for building user interfaces. It helps developers create reusable UI components and manage the application state effectively. Maintained by Meta (Facebook), it’s widely used for building modern single-page applications (SPAs).

Node.js

Node.js is a runtime environment that allows JavaScript to run on the server side. It’s used for building fast and scalable backend applications and APIs.

Git

Git is a version control system that helps developers track changes in code, collaborate with others, and manage different versions of a project efficiently. It’s essential for team-based development.

Microsoft Excel

Microsoft Excel is a powerful spreadsheet application used for data organization, analysis, and visualization. It's widely utilized in business, finance, engineering, and data analysis for managing information efficiently.
